.. _Create a Session:

**************************
Create a Session
**************************

.. autoclass:: sessions.views.SessionsList
    :members:

**Example post**

.. code-block:: json

    {
         "username": "name",
         "password": "password"
    }

**Example response**

.. code-block:: json

    HTTP 201 CREATED
    Vary: Accept
    Content-Type: text/html; charset=utf-8
    Allow: POST, OPTIONS

        "expires": 604800,
        "user": {
            "id": 4,
            "email": "staff@example.com",
            "username": "staff",
            "first_name": "",
            "last_name": "",

            "organizations": []
        },
        "uri": "http://localhost:8000/api/sessions?username=staff&password=edx/938680977d04ed091b67b974b6b6be60"
    }
